# Воронка / Watch folder

Бросьте сюда аудио/видео или готовую транскрипцию (JSON/TXT) — файл
автоматически уйдёт в обработку, а потом переместится в `processed/`
(успех) или `failed/` (ошибка).

Категорию можно задать подпапками:

    inbox/<Категория>/<Подкатегория>/запись.mp3

Если бросить файл прямо в корень `inbox/`, используются значения по
умолчанию из .env (WATCH_DEFAULT_CATEGORY / WATCH_DEFAULT_SUBCATEGORY).

Поддерживаемые форматы: mp3, wav, m4a, ogg, mp4, mkv, flac, aac, webm,
а также .json (WhisperX) и .txt (*_speakers) — для пропуска транскрибации.
